# 用 Excalidraw 在 Obsidian 里画流程图、架构图：从入门到嵌入笔记

> Excalidraw 是 Obsidian 社区最热门的绘图插件之一（GitHub 6.7k Star），支持手绘风格的流程图、架构图、思维导图和示意图。文章覆盖从新建绘图文件、基础图形工具使用、嵌入 Markdown 笔记，到创建内部链接、嵌入其他内容、导出 PNG/SVG，以及三个实战场景（项目流程图、系统架构图、思维导图）。

![[assets/excalidraw-obsidian-1.png]]

## Excalidraw 能画什么

Excalidraw 是一个手绘风格的绘图工具，嵌入 Obsidian 后，你能用它画：

- 流程图（业务流程、工作流程）
- 架构图（系统架构、技术架构）
- 思维导图（知识结构、想法梳理）
- 示意图（概念解释、关系说明）

它的特点是"手绘风格"——线条不是完美的直线，框框不是完美的矩形，看起来像是手画出来的。这种风格有两个好处：

- 画的时候不纠结细节，能快速表达想法
- 读者看的时候不会觉得是"死板的官方文档"

GitHub 上这个插件有 6.7k Star，是 Obsidian 社区里最热门的绘图插件之一。

## 怎么新建一张 Excalidraw 图

在 Obsidian 里新建 Excalidraw 图。新建的文件后缀是 `.excalidraw.md`，它既是 Excalidraw 图，也是 Markdown 文件。这意味着它能被 Obsidian 搜索、能被双链引用、能出现在反向链接面板里。

![[assets/excalidraw-obsidian-4.png]]

## 基础绘图：图形工具和手绘风格

打开文件后，你会看到一个绘图界面。

### 图形工具

左侧工具栏有这些工具：

- 矩形/椭圆/菱形：画框框，用来表示步骤、节点、分类
- 箭头/线条：画连接线，表示流程方向、关系
- 文本：写文字，放在框框里或旁边
- 手绘工具：画自由线条，适合画示意图
- 橡皮擦：删除元素

用法很简单：选中工具，在画布上拖动就能画。画完后可以调整大小、位置、颜色。

### 手绘风格

Excalidraw 的"手绘风格"是自动的——你画一个矩形，它会自动变成"手绘效果"的矩形。如果你想让线条更"手绘"或更"规整"，可以调整：

- 选中元素后，右侧面板有 Roughness 选项
- 0 是规整线条，1-5 是不同程度的手绘效果
- 默认是 1，轻微手绘风格

### 颜色和样式

右侧面板还能调整：

- 填充颜色：框框内部的颜色
- 线条颜色：边框的颜色
- 线条粗细：边框的粗细
- 透明度：填充的透明度

建议：用颜色区分不同类型的节点（比如"输入"用蓝色、"处理"用绿色、"输出"用橙色），但不要用太多颜色，2-3 种就够了。

## 嵌入 Markdown 笔记的几种方式

Excalidraw 图画完后，你可能想把它嵌入其他笔记里。

### 基础嵌入

在 Markdown 笔记里，用这种语法嵌入：

```
![[文件名.excalidraw]]
```

比如你有一张图叫 `业务流程图.excalidraw.md`，在另一篇笔记里写：

```
![[业务流程图.excalidraw]]
```

这张图就会显示在笔记里。

### 强制显示图片

有一种特殊情况：如果你在图的 frontmatter 里设置了 `excalidraw-embed-md: true`，嵌入时会显示 Markdown 内容而不是图片。这时候如果你想强制显示图片，可以用特殊语法：

```
![[图.excalidraw#^as-image]]
```

`#^as-image` 是一个"幻影 block reference"，会强制渲染图片。

## 在图里创建链接

Excalidraw 图不只是"画完就放着"，它也能包含链接。

### 链接到其他笔记

选中一个文本元素，输入：

```
[[另一篇笔记]]
```

这个文本就会变成链接，点击后跳转到那篇笔记。这意味着：你的流程图不只是"可视化"，还能作为"导航"——点击某个节点，跳转到详细说明。

### 链接到图里的特定元素

你可以用 block reference 链接到图里的特定元素：

| 语法 | 效果 |
|------|------|
| `[[图#^elementID]]` | 转clude 该元素的文本 |
| `[[图#^group=elementID]]` | 选中该元素所在的组 |
| `[[图#area=Section标题]]` | 显示该区域的截图 |

这些链接会出现在反向链接面板里，和普通笔记一样。

### 外部链接

也能加外部链接：

```
https://itkee.cn
[Obsidian](https://itkee.cn)
```

如果你从浏览器拖一个 YouTube 链接到 Excalidraw，它会变成一个缩略图+链接的形式。

## 嵌入其他内容到图里

Excalidraw 图不只是"画图"，还能嵌入其他内容。

### 嵌入 Markdown 文件

从 Obsidian 文件浏览器拖一个 Markdown 文件到画布上，按住 Shift 键再放下，这个 Markdown 文件就会嵌入到图里。也可以用命令面板：`Excalidraw: Insert markdown file from vault`。嵌入的 Markdown 文件会显示为一个小卡片，可以设置字体和样式。

### 嵌入图片

能从文件浏览器拖图片，也能直接粘贴图片。图片会保存在你的 Vault 里。如果你从网页拖图片时按住 Ctrl，图片不会保存到 Vault，只会保留 URL 链接。

### 嵌入 LaTeX 公式

用命令面板：`Excalidraw: Insert LaTeX formula`。输入 LaTeX 代码，公式会渲染到图里。想修改公式，可以用 Ctrl/Cmd + Click 点击它。

## 导出 PNG/SVG

如果你想把图用于其他场景（比如公众号文章），可以设置自动导出。在插件设置里，找到 Embed & Export 部分，勾选 Auto-export SVG 或 Auto-export PNG。每次保存图时，会自动生成一个 PNG 或 SVG 文件，和原图放在同一目录。

## 三个实战场景

### 场景一：画项目流程图

你在做一个项目，想整理一下流程。用矩形画每个步骤（比如"需求收集"、"方案设计"、"开发实现"、"测试上线"），用箭头连接步骤，表示流程方向。用颜色区分阶段（比如"规划阶段"用蓝色、"执行阶段"用绿色）。保存后，嵌入项目笔记：

```
![[项目流程图.excalidraw]]
```

这样你的项目笔记里既有文字说明，又有可视化流程图，一目了然。

### 场景二：画系统架构图

你在整理一个技术系统的架构。用矩形/椭圆画各个组件（比如"前端"、"后端"、"数据库"、"缓存"），用箭头连接组件，表示数据流向。用菱形画"决策节点"（比如"负载均衡"、"路由判断"）。在图里创建链接：点击某个组件，跳转到详细说明文档。这样做的好处：架构图和文档在一起，更新架构时直接编辑图。

### 场景三：画思维导图梳理知识

你在学一个新领域，想梳理知识结构。在中心画一个大矩形写主题名称，周围画小矩形写子主题，用线条连接中心和子主题。用颜色区分层级（中心用深色、子主题用浅色）。这样画出来的思维导图，能嵌入你的学习笔记，帮助你复习时快速回忆整体结构。

## 和其他 Obsidian 功能配合

### 和标签配合

Excalidraw 文件也能加标签。在图的 Markdown 部分写：

```yaml
tags: [流程图, 项目管理]
```

这样这张图就能被标签搜索找到。

### 和搜索配合

Excalidraw 文件是 Markdown 文件，所以 Obsidian 的搜索功能能搜到它。你搜索"流程图"，能找到所有包含这个词的图。

### 和 Templater 配合

注意：如果你用 Templater 自动创建 Excalidraw 文件，要小心"Cannibalization"问题——Templater 可能把已同步的文件当成新文件再处理一次，导致文件损坏。解决方法：在模板开头加一段判断逻辑，检查文件是否在模板目录里。

## 什么时候用 Excalidraw

Excalidraw 适合：

- 快速画一张图，不需要完美细节
- 图要嵌入笔记，和文字放在一起
- 图要被搜索、被引用、被双链跳转
- 个人知识库、项目文档、学习笔记

如果你的需求是"在 Obsidian 里画一张图，和笔记在一起"，Excalidraw 是合适的选择。

## 总结

Excalidraw 插件解决的问题很简单：在 Obsidian 里直接画图，不用打开其他软件，就能画出好看的手写风格插图。如果你已经在用 Obsidian 做知识库，需要画图的时候，可以试试 Excalidraw——它能让你的笔记不只是文字，还有可视化内容，美观漂亮。
